# HanBLAS

HanBLAS is a toy BLAS library implemented in rust.(WIP)



## Supported functions

### level1
- sasum, dasum, scasum, dzasum
- sdot, ddot, dsdot
- cdotu, zdotu
- cdotc, zdotc
- scopy, dcopy, ccopy, zcopy
- isamax, idamax, icamax, izamax
- isamin, idamin, icamin, izamin
